# Cognizant .NET Full Stack Engineer (FSE) Learning Program

## Overview

This repository contains the solutions, assignments, and hands-on labs completed as part of the **Cognizant .NET Full Stack Engineer (FSE) Learning Program**. The projects are designed to provide practical experience with modern .NET technologies, backend development, database management, web development, and software engineering best practices.

The repository is organized module-wise, with each folder representing a specific lab, assignment, or learning exercise completed during the program.

## Getting Started

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Open the desired project in Visual Studio or VS Code.

3. Restore NuGet packages.

```bash
dotnet restore
```

4. Build the project.

```bash
dotnet build
```

5. Run the project.

```bash
dotnet run
```

For Web API projects, Swagger UI will be available in the browser when running in the Development environment.
